You MUST follow these 5 steps before downloading McAfee because Windows Defender cannot be enabled simultaneously with McAfee. It will make McAfee ineffective if not done correctly.

1. Press Windows Key + I to open Settings.

2. Navigate to Privacy & Security > Windows Security, then click Virus & threat protection.

3. Under Virus & threat protection settings, click Manage settings.

4. Toggle the Real-time protection, Cloud-Delivered protection, Automatic sample submission and Tamper Protection switches to Off.

5. Click Yes on the User Account Control (UAC) prompt.

Once you have done these 5 steps, you can safely download and install McAfee. Once McAfee is installed and running, you can restore your Windows Defender settings if you wish, but it is not recommended. To run McAfee, simply double-click the icon in your system tray, and the real time protection will be active.
